
 Instrument Showroom Management System (Python)

Description

This project is a Python Tkinter GUI application designed for managing a musical instrument showroom. It allows users to select instruments, enter quantities, apply discounts, calculate the total bill dynamically, and store all order details securely in a MySQL database.

Objectives

* Connect to a MySQL database to manage instrument and order records
* Display available musical instruments with prices in a Tkinter GUI
* Allow users to select instruments and specify quantity
* Validate user input and handle errors
* Calculate the total bill dynamically
* Support adding, updating, deleting, and viewing orders

Technologies Used

* Python
* Tkinter
* MySQL
* MySQL Connector

Project Modules

* GUI Module
* Database Connectivity Module
* Billing Module
* Order Management Module

Conclusion

This project demonstrates the integration of a MySQL database with a Python Tkinter GUI to manage musical instrument sales efficiently. It supports key operations such as adding, updating, deleting, and viewing records, while providing a simple and interactive user interface.
